Permissions & what they’re used for

PromptProtect requests browser permissions only to enable its core functionality.

  • storage — store your settings and allow rules locally.
  • tabs — coordinate the dashboard UI and active tab context.
  • host access — run detection UI on supported sites.
  • side panel / sidebar — show the dashboard (Chrome side panel / Opera sidebar panel).

Privacy policy

In short: PromptProtect runs locally in your browser and is designed to avoid sending your prompt contents to any external server.

  • Data collection: No analytics or ad tracking is built in.
  • Prompt contents: Detection happens locally; the extension does not intentionally transmit your prompt text off-device.
  • Local storage: Settings and rules are stored via browser extension storage.
  • Third parties: No selling or sharing of your prompt contents.

Note: When you use AI chat websites, those sites may send your text to their own servers. PromptProtect helps you catch sensitive information before you send it.
